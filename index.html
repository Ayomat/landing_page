Liberti - A virtual fashion website

The screenshot on the left is from a locally hosted website I made for a virtual fashion company called "Liberti".
The project was a full package containing a user website and an admin panel for it too.

What did it do and what did I make it with?

On the left is the admin panel which allowed the admin to view analytics such as recent orders, number of each product left in stock, best selling categories etc. It also allowed them to quickly add, view, update and delete products from the inventory.

This project was made using: PHP, MySQL, JavaScript, HTML/CSS
